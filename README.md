## 🧠 Project Overview

**UTM Transport** is a proposed system designed to improve transportation management within Universiti Teknologi Malaysia (UTM).

The system aims to enhance the efficiency, accessibility, and user experience of campus transportation services by providing better scheduling, tracking, and management features for students.

---

## ✨ Features


* Login Authentication
* Live Bus Tracking
* Seat availability Monitor 
* Carpool
* Smart Route Planning
* Feedback & Rating
